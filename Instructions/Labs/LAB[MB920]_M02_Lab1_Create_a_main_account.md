# 學習路徑 2：學習 Microsoft Dynamics 365 Finance 的基本概念
# 課程模組 2：描述一般總帳

## 實驗室 2.1：建立主要帳戶

## 實驗設定

   - **預估時間**：5 分鐘

## 目標

在此實驗室中，您將執行下列活動：

1. 建立法律實體。

2. 建立會計行事曆。

3. 建立帳戶圖表。

4. 在帳戶圖表中建立主要帳戶。

5. 建立部門和成本中心類型的作業單位。

6. 使用主要帳戶和成本中心建立會計結構。

7. 設定總賬。

# 實驗室步驟

## 建立法律實體

1. 在 Dynamics 365 Finance 的左側瀏覽窗格中，選取 **[模組********&gt;組織管理****&gt;組織&gt;法律實體]。**

2. 在 [ **法律實體]** 頁面上，選取 **動作窗格中的 [新增** ] 按鈕以建立新的法律實體，然後輸入下列資訊：

    - 名稱： **Contoso 示範系統**

    - 公司： **CDS**

    - 國家/地區： **美國**

3. 選取底部的 [ **確定** ]。

## 建立會計行事曆

1. 在Dynamics 365 Finance 的左側流覽窗格中，選取 **[模組&gt;******** 一般總帳****&gt;行事曆][會計行事歷]。&gt;** 

2. 在 [ **會計行事曆]** 頁面上，選取 **動作窗格中的 [新增行事曆** ] 按鈕以建立新的會計行事曆，然後輸入下列資訊：

    - 行事曆： **CDS**

    - 描述： **示範系統行事曆**

    - 會計年度開始時間： **2024 年 9 月 30 日**

    - 會計年度結束： **2024/10/1**

    - 會計年度名稱： **2024-25**

    - 期間長度： **1**

    - 單位： **月**

3. 選取**建立**按鈕。

系統會產生14個期間，包括一個開啟和一個結束期間，以及12個月的12個期間。

## 建立帳戶圖表

1. 在 Dynamics 365 Finance 的左側瀏覽窗格中，選取 **[帳戶] 的&gt; ****&gt; [模組**** > 一般總帳******&gt;** 圖表]。**

2. 在 [ **帳戶** 圖表] 頁面上，選取 **動作窗格中的 [新增** ] 按鈕以建立新的帳戶圖表，然後輸入下列資訊：

    - **帳戶**圖表：示範系統

    - **描述**：Contoso 示範系統

3. 選取 [主要帳戶** FastTab] 上的 **[**新增**] 按鈕，以使用下列值建立主要帳戶：

    - 主要帳戶： **1000**

    - 名稱： **現金**

    - 主要費用類型： **資產負債表**

    - 主要帳戶類別： **CASH**

    - DB/CR 預設：**借方**

4. 建立另一個主要帳戶：

    - 主要帳戶： **3000**

    - 名稱： **營收**

    - 主要費用類型： **營收**

    - 主要帳戶類別： **REV**

    - DB/CR 預設值： **點數**

5. 建立另一個主要帳戶：

    - 主要帳戶： **6000**

    - 名稱： **旅行費用**

    - 主要費用類型： **費用**

    - 主要帳戶類別： **EXP**

    - DB/CR 預設：**借方**

##  建立營運單位

1. 流覽至 **[模組&gt;******** 組織管理&gt;****組織&gt;] 作業單位。**

2. 在動作窗格中，選取 **[新增** ] 按鈕， **後面接著 [部門作業** 單位類型]，然後輸入下列值：

    - 名稱： **CDS_Training**

3. 在動作窗格中，選取 [新增] 按鈕，後面接著 [成本中心作業單位類型]，然後輸入下列值：

    - 名稱： **CDS_Purchase**

4. 再新增兩個成本中心： **CDS_IT** 和 **CDS_Admin**。

## 建立會計結構

1. 在 [**財務與作業首頁**] 的右上方，確認您正在與 CDS** 公司合作**。

2. 如有必要，請選取公司，然後從功能表中選取 **CDS**。

3. 在 [Dynamics 365 Finance] 的左側瀏覽窗格中，選取 **[帳戶結構的&gt;模組**** > 一般總帳****&gt;****圖表] [設定帳戶&gt;結構]。**

4. 在動作窗格中，選取 [ **新增** ] 按鈕，以使用下列值建立新的帳戶結構：

    - 帳戶結構： **CDS_BS**

    - 描述： **CDS 資產負債表**

    - 新增主要帳戶： **是**

5. 選取**建立**按鈕。

6. 在 [ **區段] 和 [允許的值** FastTab] 上，選取 [ **新增** ] 按鈕以新增 **主帳戶範圍 1000..2999**。

7. 選取動作窗格中的 [ **啟動** ] 按鈕，後面接著 **批處理對話方塊中的 [啟動** ] 按鈕。

8. 在 [帳戶結構] 頁面的 **[動作] 窗格中，選取 [**新增**] 按鈕，以使用下列值建立另一個帳戶**結構：

    - 帳戶結構： **CDS_Revenue**

    - 描述： **CDS 營收**

    - 新增主要帳戶： **是**

9. 選取**建立**按鈕。

10. 在 [ **區段] 和 [允許的值** FastTab] 上，選取 [ **新增** ] 按鈕以新增 **主帳戶範圍 3000..5999**。

11. 選取動作窗格中的 [ **啟動** ] 按鈕，後面接著 **批處理對話方塊中的 [啟動** ] 按鈕。

12. 在 [帳戶結構] 頁面的 **[動作] 窗格中，選取 [**新增**] 按鈕，以使用下列值建立另一個帳戶**結構：

    - 帳戶結構： **CDS_Expense**

    - 描述： **CDS 費用**

    - 新增主要帳戶： **是**

13. 選取**建立**按鈕。

14. 在 [ **區段] 和 [允許的值** FastTab] 上，選取 [ **新增** ] 按鈕以新增 **主帳戶範圍 6000..9999**。

15. 選取區段和允許值** FastTab 上的 **[**區段**] 按鈕

16. 在 [**新增區段**] 對話框中，選取 [CostCenter **]，然後選取 **[**新增區段**] 按鈕。

17. 在 [ **CostCenter** ] 字段中，輸入 **253..255**。 

18. 選取動作窗格中的 [ **啟動** ] 按鈕，後面接著 **批處理對話方塊中的 [啟動** ] 按鈕。

## 配置分類帳

1. 在 [**財務與作業首頁**] 的右上方，確認您正在與 CDS** 公司合作**。

2. 如有必要，請選取公司，然後從功能表中選取 **CDS**。

3. 在 Dynamics 365 Finance 的左側瀏覽窗格中，選取 **[模組 > ****一般總帳設定&gt;總帳&gt;] 頁面**並設定下列專案：

    - 帳戶圖表： **示範系統**

    - 會計行事曆： **CDS**

    - 會計貨幣： **美元**

    - 報告貨幣： **美元**

    - 會計貨幣匯率類型： **預設值**

    - 預算匯率類型： **預算**

4. 在 [ **帳戶結構** FastTab] 上，選取 [ **新增** ] 按鈕以新增帳戶結構 **CDS_BS**。

5. 再新增兩個帳戶結構： **CDS_Revenue** 和 **CDS_Expense**。

這會完成一般總賬模組的基本設定。 您現在可以規劃日誌設定以張貼日誌。
